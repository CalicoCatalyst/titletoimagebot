# Title2ImageBot

Fork of Original bot by gerenook. 

Depends: 

> Praw    
> Pillow  
> PyImgur  
> ImgurPython  
> argparse  

Syntax:

> bot.py \<items> <time (s)>

So for example, running `python bot.py 10 60` Would have the bot check the 10 most recent inbox items every 60 seconds and respond to them.
